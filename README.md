# Tic_Tac_Toc_Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project provides a clean and responsive user interface where two players can play the classic Tic Tac Toe game in their web browser.

## Features

* 🎮 Two-player gameplay (Player X vs Player O)
* ✨ Interactive and user-friendly interface
* 🏆 Automatic winner detection
* 🔄 Restart/New Game button

## Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling, layout, and responsive design
* **JavaScript (ES6)** – Game logic, player turns, winner checking, and reset functionality

## How to Play

1. Open the game in your web browser.
2. Player **X** starts the game.
3. Players take turns clicking on empty cells.
4. The first player to get **three symbols in a row, column, or diagonal** wins.
5. If all cells are filled without a winner, the game ends in a draw.
6. Click the **Restart** or **New Game** button to play again.
